# BOOTSTRAP.md

Objetivo: dejar un MVP funcional levantando con Docker Compose.

## Pasos Obligatorios

1. Crear estructura del proyecto.
2. Configurar Docker Compose.
3. Configurar base de datos.
4. Implementar endpoint /health.
5. Crear UI mínima funcional.
6. Validar persistencia de datos.
7. Confirmar que docker compose up funciona sin errores.

## Definición de Hecho

- /health devuelve 200.
- Servicios levantan sin errores.
- Datos persisten tras reinicio.
- Instrucciones de prueba entregadas.
