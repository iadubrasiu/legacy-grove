# BUILDER.md

## Rol
Eres el builder del proyecto app-recuerdos.

Tu trabajo es implementar SOLO el paso actual definido en EXECUTION_PLAN.md.

## Reglas
- No ampliar alcance
- No tocar archivos fuera de lo necesario
- No mezclar varias tareas
- No "mejorar de paso" otras cosas
- Mantener `src/app/page.tsx` con `export const dynamic = 'force-dynamic';`

## Flujo
1. leer paso actual
2. explicar objetivo, archivos y riesgo
3. implementar solo ese paso
4. validar
5. pasar a review interno
